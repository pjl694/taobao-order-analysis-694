# taobao-order-analysis-694
# 淘宝订单数据分析

## 项目简介
使用 SQL（窗口函数、分组聚合）和 Excel 对 2.8 万+ 条订单数据进行多维度分析，输出可落地的运营建议。

## 数据说明
- 数据量：28,010 条订单  
- 时间范围：2020年2月  
- 主要字段：订单金额、省份、创建时间、付款时间、退款金额等

## 文件说明
- `sql/` ：8 个 SQL 查询脚本  
- `data/` ：导出 CSV 结果  
- `报告/` ：最终分析报告 PDF

## 核心分析内容
1. 各省份销售额排名  
2. 每小时订单量及销售额  
3. 累计销售额趋势  
4. 金额分层统计  
5. 各省份退款率排名  
6. 各小时退款订单数及退款率  
7. 未付款订单按小时分布  
8. 未付款订单按省份分布

## 主要结论
- 销售高峰集中在 20-23 点，上海、北京、江苏贡献 45% 销售额。  
- 湖北、新疆等省份退款率超过 50%，需重点优化物流。  
- 凌晨未付款订单占比高，建议次日短信提醒+优惠券。

## 使用工具
- SQL：datagrip（窗口函数、CTE、子查询）  
- Excel：数据透视表、双轴图、条形图
